# --Repository-name-restaurant-order-analysis-SQL
You have been hired as a Data Analyst for the "Taste of World Cafe". The restaurant has debuted a new menu and you have been asked to analyse the data and get some information

Dataset

The dataset is sourced from Maven Analytics. I’m grateful to Maven Analytics for providing such a valuable opportunity to practise and develop my SQL analysis skills through this project.

Skills showcased in this Project

Use of SELECT and WHERE statements along with LIMIT and ORDER BY
Use of Aggreagte Functions such as SUM, MAX, MIN, COUNT
Use of GROUP BY and JOINS
Use of Subquery

Queries
Objective 1 : Explore the Menue items table

View the menu_items table and write a query to find the number of items on the menu
